# 创意厨房预约系统 PRD

## 产品定位

创意厨房预约系统用于社区成员按微信名预约公共厨房设备。普通用户无需注册登录，只需输入社区微信群内昵称即可查看、预约、取消自己的预约。

## 用户角色

- 普通用户：输入微信名后使用首页、预约页、我的预约、使用规则。
- 管理员：知道管理密码的人可访问 `/admin` 登录，或在已登录管理员状态下从首页右上角用户菜单进入管理后台。

## 核心功能

- 首页按日期展示风炉、平炉、发酵箱等设备的时间条。
- 时间条支持 06:00-24:00 展示，22:00 之后的格子仅可作为结束时间，不可作为开始时间。
- 每个设备卡片提供轻量“预约”按钮，进入当前所选日期的设备预约页。
- 点击首页整条设备时间轴即可进入预约页，系统按点击位置计算对应开始时间；点到不可预约格子时进入详情页但不预选开始时间。
- 首页每个设备时间条右上方展示图例：可预约、已预约、我的预约、不可用。
- 首页不再显示绿色“可用”设备状态，仅在设备维护时显示“维护中”。
- 我的预约按日期分组；同一天内先按设备聚合，再按开始时间排序。
- 用户可取消自己尚未开始的预约。
- 用户可在首页菜单中修改本机当前微信名；已有预约记录仍保留原名称，不会自动改名。
- 管理员可查看预约、强制取消、封禁/解封用户、维护/恢复设备、调整预约配置。

## 预约规则

- 用户维度为微信名 `wechat_name`。
- 单次最大时长、单设备单日最大预约次数、预约窗口、时段粒度、营业时间由后台配置。
- 当前默认配置：单次最长 2 小时、每设备每天最多 2 次、时段粒度 30 分钟、06:00 开始、24:00 结束。
- 数据库通过排他约束阻止同一设备同一天的有效预约时间重叠。
- 被封禁用户不能新增预约。

## 技术实现

- 前端：Next.js App Router、React、Tailwind CSS。
- 数据库：Supabase Postgres。
- 管理模式：轻量密码登录，适合小范围内部使用；不要求普通用户鉴权。
- Lint：使用 ESLint flat config 和 `eslint-config-next`，`npm run lint` 可正常执行。
